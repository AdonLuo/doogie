Overview

Doogie is a browser based on Chromium (via CEF) and built using Qt. The primary goal is to make a browser that functions more like an IDE. In an era where web browsers continually remove or hide UI, Doogie aims to make it more visible and allow easier cross-contextual browsing. Read the user guide

Not Yet Implemented

Many features are either not yet implemented or will never be implemented. See the GitHub issues. Some of the major ones are:

  • macOS support
  • HTTP basic and client certificate authentication
  • Automatic updates
  • Cosmetic element hiding based on blocker rules
  • Flash
  • Form resubmission
  • Microphones and webcams
  • Geolocation
  • Notifications
  • Plugins
  • Extensions
... and much much more. Contributions welcome.
